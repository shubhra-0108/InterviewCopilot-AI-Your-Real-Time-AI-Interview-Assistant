## 🧠 What is InterviewCopilot AI?

InterviewCopilot AI is a smart assistant designed to help you during coding interviews, technical rounds, and practice sessions.

It listens, analyzes, and generates intelligent responses in real-time — acting like your personal interview coach.

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/interview-copilot-ai.git
cd interview-copilot-ai
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
```

Create `.env` file:

```
PORT=5000
API_KEY=your_api_key_here
```

Run server:

```
npm run dev
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm run dev
```
